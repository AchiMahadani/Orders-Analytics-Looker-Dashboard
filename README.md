# 📊 Orders Report Dashboard | Google Looker Studio

An interactive **Orders Analysis Dashboard** built using **Google Looker Studio** to visualize and analyze order data across different dimensions such as region, segment, category, customer, and time trends.

---

## 🚀 Project Overview

This project focuses on transforming raw order data into meaningful business insights through interactive dashboards and visual storytelling. The dashboard enables users to explore order performance, customer distribution, and geographical trends for better decision-making.

## 🎯 Objectives

- Analyze order performance across different regions.
- Understand customer segmentation and order distribution.
- Identify category-wise and city-wise order patterns.
- Track order trends over time.
- Build an interactive dashboard for business intelligence reporting.

## 📊 Dashboard Features

### KPI Cards
- Total Orders: **5,009**
- Total Customers: **793**
- Total States: **49**
- Total Segments: **3**

### Interactive Filters
- Region Filter
- Segment Filter
- Category Filter
- Ship Mode Filter

### Visualizations
- Orders by Region
- Orders by Category
- Orders by Segment
- Orders by City
- Year-wise Order Trend
- Month-wise Order Trend
- Order Details Table
- State-wise Quantity Analysis
- Customer-wise Quantity Analysis

## 📈 Key Insights

- Consumer segment generated the highest number of orders.
- Office Supplies contributed the largest share among categories.
- Order quantity varied significantly across regions and states.
- Seasonal trends can be observed from monthly order analysis.
- Interactive filters allow users to drill down into specific business areas.
